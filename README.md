# Gilded Rose Refactoring Kata
Kata description - https://github.com/emilybache/GildedRose-Refactoring-Kata

# Requirements
I used:
- `ruby 3.2.2`
- `rspec 3.12`

But it should work for any reasonable `ruby` and `rspec` versions.

# Installation
1) Clone repository
2) `cd gilded_rose_kata`
3) `rspec spec`
